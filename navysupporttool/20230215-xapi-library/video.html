How to Track Video Content

To get started with implementing xAPI in your video content, check out the resources below.

Navy Core xAPI Profile

The Navy Core xAPI Profile defines the requirements for all Navy xAPI implementations regardless of content type or other tracking requirements.

  • Profile

Navy Common Reference Profile

The Navy Common Reference Profile provides the rules, xAPI Statement requirements, and Statement examples for tracking various types of general Activities, including video interactions. Click the download proifle button below to view the documentation.

  • Profile

Video Player Example & Code Library

The Video Example uses the Video Player Code Library below and demonstrates how a video content implements xAPI to track video interactions such as pausing, playing, seeking, completing a video. Click the download example button below to "look under the hood" and see how it works.
